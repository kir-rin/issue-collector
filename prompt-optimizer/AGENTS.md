# Project Instructions

## Documentation

When you need to search docs, use `context7` tools.

## Troubleshooting

새로운 에러를 해결했을 때, `airflow/TROUBLESHOOTING_TEMPLATE.md`를 참고하여 `airflow/TROUBLESHOOTING.md`에 기록하세요.

### 작성 순서

1. **증상 파악**: 에러 로그나 문제 상황을 그대로 복사
2. **발견 방법**: 어떤 커맨드로 문제를 발견했는지 기록
3. **원인 분석**: 왜 문제가 발생했는지 설명
4. **해결 방법**: 실행 가능한 코드나 설정을 작성
5. **참고 문서**: (선택) 공식 문서나 관련 링크
